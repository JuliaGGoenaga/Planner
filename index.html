GitHub Gist sync

Your data is saved as JSON in a private Gist and synced across all devices.

  1. Go to github.com/settings/tokens/new → choose Classic token → tick only gist → Generate.
  2. Go to gist.github.com → New secret Gist → filename planner.json → content {} → Create.
  3. Copy the Gist ID from its URL: gist.github.com/you/ID_HERE
Stored only in this browser — sent only to api.github.com.

External calendars (ICS)

Paste the secret ICS link from Google Calendar or Outlook. Events appear read-only in week and month views.

  1. Google: calendar.google.com → Settings → your calendar → Secret address in iCal format
  2. Outlook.com: Settings → Calendar → Shared calendars → Publish → copy ICS link
Tries direct fetch + 5 free public proxies automatically. No login required.

Task rollover

To-dos not completed before today are automatically moved to today on app load. They show a ↩ badge and keep their original date as reference.

ICS & CORS note

Browsers block direct ICS fetches. The app first tries a direct request (works with self-hosted or CORS-enabled calendars), then falls back through 5 free public proxies automatically. No login needed. If all fail, deploy the Netlify proxy below.

CORS Proxy — Netlify Function

Deploy a personal proxy on Netlify's free tier (125k req/month). The app creates the site and uploads the function automatically — no CLI, no credit card required. The function code is also saved to your Gist as proxy.js.

  1. Go to app.netlify.com → User settings → Personal access tokens → New access token.
  2. Paste the token below and click Deploy proxy. The app will create a Netlify site and deploy the function in one click.
Stored in this browser only — never uploaded to your Gist.
