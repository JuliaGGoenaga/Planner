GitHub Gist sync

Your data is saved as JSON in a private Gist and synced across all devices.

  1. Go to github.com/settings/tokens/new → choose Classic token → tick only gist → Generate.
  2. Go to gist.github.com → New secret Gist → filename planner.json → content {} → Create.
  3. Copy the Gist ID from its URL: gist.github.com/you/ID_HERE
Stored only in this browser — sent only to api.github.com.

External calendars (ICS)

Paste the secret ICS link from Google Calendar or Outlook. Events appear read-only in week and month views.

  1. Google: calendar.google.com → Settings → your calendar → Secret address in iCal format
  2. Outlook.com: Settings → Calendar → Shared calendars → Publish → copy ICS link
Uses public CORS proxies to fetch your calendar. If one fails, others are tried automatically.

Task rollover

To-dos not completed before today are automatically moved to today on app load. They show a ↩ badge and keep their original date as reference.

ICS & CORS note

Browsers block direct ICS fetches. The app tries three public proxies automatically. Your ICS URLs are secret links — not credentials. If all proxies fail, the sync status bar will show an error.
