GitHub Gist sync

Your planner data can sync to a private GitHub Gist. This version validates the token and Gist before enabling sync.

  1. Create a classic GitHub token with only the gist scope. Fine-grained tokens often do not work with Gists.
  2. Create a secret Gist with filename planner.json and content {}.
  3. Copy the Gist ID from the URL, not the full URL.
Stored only in this browser. If sync fails, your edits remain local until the connection works.

Google Calendar

Not connected

Pega la URL privada en formato iCal/ICS para mostrar tus eventos en semana y mes. La integración es solo lectura.

  1. Abre calendar.google.com → ⚙️ Settings → tu calendario → Integrate calendar.
  2. Copia la Secret address in iCal format.
  3. Pega aquí la URL que empieza por https://calendar.google.com/calendar/ical/...
Se guarda solo en este navegador. No hace falta OAuth ni Google Cloud.

Outlook / Microsoft 365

Not connected

Pega la URL ICS publicada de Outlook para mostrar tus eventos. La integración es solo lectura.

  1. Abre outlook.live.com → Settings → View all → Calendar → Shared calendars.
  2. En Publish a calendar, elige tu calendario y copia el enlace ICS.
  3. Pega aquí la URL publicada.
Para cuentas de empresa puede que el administrador limite la publicación ICS.

Task rollover

Any to-do that was not completed before today is automatically moved to today when you open the app. You'll see a small rolled over badge on those tasks.

This happens silently in the background — your data is saved immediately to Gist after rollover so it persists across devices.
