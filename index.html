Sincronización con GitHub Gist

  1. Ve a github.com/settings/tokens/new → elige Classic token → marca solo gist → Generate.
  2. Ve a gist.github.com → New secret Gist → nombre de archivo planner.json → contenido {} → Create.
  3. Copia el ID del Gist desde su URL: gist.github.com/tu_usuario/ID_AQUÍ
Se guarda solo en este navegador — solo se envía a api.github.com.

Calendarios externos (ICS)

  1. Google: calendar.google.com → Ajustes → tu calendario → Dirección secreta en formato iCal
  2. Outlook.com: Ajustes → Calendario → Calendarios compartidos → Publicar → copia el enlace ICS
Si la URL es muy larga (Outlook) y falla con error 413, usa el botón Acortar antes de añadir — los tokens de autenticación se conservan.

Asistente IA

  1. Crea cuenta gratuita en console.groq.com (no requiere nada instalado, funciona desde cualquier dispositivo).
  2. Ve a API Keys → Create API Key → copia la clave (empieza por gsk_).
  3. Pégala abajo y pulsa Guardar. Aparecerá el botón flotante en todas las pestañas.
  4. Portátil de empresa: deja la clave vacía (sin asistente) o pon la URL del LLM interno en el campo Endpoint.
Guardada solo en este navegador — nunca va al Gist ni a ningún servidor salvo al enviar una pregunta.
Cámbialo si tu empresa tiene su propio LLM compatible con la API de OpenAI.

Sincronización con GitHub Gist

Tus datos se guardan como JSON en un Gist privado y se sincronizan en todos tus dispositivos.

Calendarios externos (ICS)

Pega el enlace secreto ICS de Google Calendar u Outlook. Los eventos aparecen de solo lectura en las vistas semanal y mensual.

Arrastre de tareas

Las tareas no completadas antes de hoy se mueven automáticamente a hoy al cargar la app. Muestran una insignia ↩ y conservan su fecha original como referencia.

Nota sobre ICS y CORS

Los navegadores bloquean las peticiones ICS directas. La app primero comprueba la caché de calendarios de la GitHub Action, luego intenta una petición directa (funciona con calendarios propios o con CORS habilitado), y por último recurre a proxies públicos gratuitos automáticamente. No requiere iniciar sesión.
