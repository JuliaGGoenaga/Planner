GitHub Gist sync

Your data is saved automatically to a private GitHub Gist and synced across all devices.

  1. Go to github.com/settings/tokens/new, create a token with gist scope.
  2. Go to gist.github.com, create a secret Gist with filename planner.json and content {}.
  3. Copy the Gist ID from its URL.
Stored only in your browser — sent only to GitHub.

Google Calendar

Not connected

Connect to show your Google Calendar events in week and month views (read-only).

  1. Go to console.cloud.google.com → New project.
  2. Enable Google Calendar API.
  3. Create OAuth 2.0 Client ID (Web app). Add your GitHub Pages URL as Authorised JavaScript origin AND as Authorised redirect URI.
  4. Copy the Client ID below.
Must match the origin configured in Google Cloud Console.

Outlook / Microsoft 365

Not connected

Connect to show your Outlook / Microsoft 365 calendar events (read-only).

  1. Go to portal.azure.com → Azure Active Directory → App registrations → New registration.
  2. Set Redirect URI to: your GitHub Pages URL
  3. Under API permissions add Calendars.Read (Microsoft Graph, delegated).
  4. Copy the Application (client) ID below.
Single-tenant or multi-tenant both work.

Task rollover

Any to-do that was not completed before today is automatically moved to today when you open the app. You'll see a small rolled over badge on those tasks.

This happens silently in the background — your data is saved immediately to Gist after rollover so it persists across devices.
